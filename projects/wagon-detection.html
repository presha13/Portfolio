Wagon Detection

Automated Wagon Number Recognition System

YOLOv11 Florence OCR OpenCV CUDA

Overview

Built a real-time wagon number detection system using YOLOv11 and OpenCV. Integrated Microsoft Florence OCR for accurate text extraction from moving trains, processing video feeds to generate automated, structured CSV logs for enterprise tracking.

View Code

Key Technologies

  • Model: YOLOv11 for object detection
  • OCR: Microsoft Florence for text recognition
  • Processing: OpenCV, CUDA, GStreamer, FFmpeg
  • Output: Automated CSV logging

README.md

# Wagon Detection System ## Installation 1. Clone the repository ```bash git clone https://github.com/presha13/wagon-detection.git ``` 2. Install dependencies ```bash pip install -r requirements.txt ``` ## Usage Run the main detection script: ```bash python detect.py --source video.mp4 ``` ## Features - Real-time detection using YOLOv11 - High-accuracy OCR with Microsoft Florence - Automated CSV logging

Project Impact

This system significantly reduces manual effort in tracking railway wagons, minimizing human error and providing real-time data for logistics management. The PROD-grade pipeline ensures high efficiency and reliability in industrial environments.
