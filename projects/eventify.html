Event Management Billing Application

HTML5 CSS3 JavaScript Git

Overview

Engineered a full-featured Single Page Application (SPA) for creating, managing, and storing event management invoices. Implemented complete CRUD functionality using browser localStorage and integrated PDF generation with CSV export features.

Live Demo View Code

Key Features

  • CRUD Operations: Create, Read, Update, and Delete invoices.
  • Data Persistence: Uses LocalStorage for data retention.
  • Export Options: Generate PDF invoices and export data to CSV.
  • Responsive UI: Optimized for various device sizes.

# Eventify - Event Management Billing ## Overview Eventify is a client-side web application for generating and managing invoices for event management services. ## Features - **Invoice Generation**: Create detailed invoices with multiple items. - **Local Storage**: Auto-save drafts and history. - **Export**: Download as PDF or export to CSV. ## Getting Started 1. Open `index.html` in your browser. 2. No internet connection required (offline first). ## Tech Stack - HTML5, CSS3, Vanilla JavaScript
