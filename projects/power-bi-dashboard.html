# Student & Faculty Analytics Dashboard ## Executive Summary A comprehensive Business Intelligence solution designed to provide actionable insights into academic performance and operational efficiency. ## Key Metrics - **Attendance Rate**: Daily/Monthly tracking. - **Grade Distribution**: Department-wise analysis. - **Faculty Loading**: Teaching hours vs. administrative tasks. ## Data Pipeline 1. **Source**: MySQL Database (Student Info System). 2. **Transformation**: Power Query for cleaning and normalization. 3. **Modeling**: Star schema architecture. 4. **Visualization**: Interactive Power BI reports. ## Tools Used - Microsoft Power BI Desktop - MySQL Server - DAX (Data Analysis Expressions)
