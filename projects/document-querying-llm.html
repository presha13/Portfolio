Document Querying LLM

Hybrid Retrieval RAG System

Python RAG Ollama FAISS BM25

Overview

Designed and developed a robust local document querying application using a Hybrid Retrieval Augmented Generation (RAG) system. The solution leverages FAISS for efficient dense vector similarity search running with sentence-transformer/all-MiniLM-L6-V2 embeddings, combined with BM25 for precise sparse keyword matching. This hybrid pipeline is connected to the locally hosted Mistral LLM via Ollama to generate context-aware, highly accurate responses without relying on external cloud APIs.

Key Features

  • Hybrid Retrieval: Combines dense (FAISS) and sparse (BM25) retrieval methods for optimal context extraction.
  • Local LLM Inference: Utilizes Ollama running Mistral locally for fast, private, and secure generation.
  • Advanced Embeddings: Features the sentence-transformers/all-MiniLM-L6-V2 model for high-quality semantic understanding.
  • Document Processing: Automated ingestion, chunking, and indexing of various document types.

# Document Querying LLM (Hybrid RAG) ## Overview A powerful, privacy-first local document querying system implementing a sophisticated Hybrid Retrieval Augmented Generation (RAG) pipeline. This project allows users to chat with their documents contextually by retrieving the most relevant information using both semantic significance and keyword relevance. ## Architecture Pipeline 1. **Document Ingestion**: Parsing text from PDFs, TXT, or other documents. 2. **Chunking & Indexing**: Splitting text into chunks and computing embeddings using `all-MiniLM-L6-v2`. 3. **Hybrid Search Strategy**: - **Dense Retrieval**: Searching vector space with FAISS to find semantically similar chunks. - **Sparse Retrieval**: Using BM25 algorithmic search for exact keyword match retrieval. - **Reciprocal Rank Fusion**: Reranking results from both retrievers to provide the absolute best context to the LLM. 4. **Text Generation**: The integrated Mistral model (hosted via Ollama) reads the retrieved context and generates a precise answer. ## Tech Stack - **LLM**: Local Mistral via Ollama - **Embeddings**: `sentence-transformers/all-MiniLM-L6-v2` - **Vector Store**: FAISS - **Keyword Search**: BM25 - **Language**: Python
