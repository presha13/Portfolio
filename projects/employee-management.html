Employee Management System

Developed for Xmatiq Technologies

Next.js Node.js MySQL TailwindCSS

Overview

Designed and developed a comprehensive Employee Management System for Xmatiq Technologies to streamline HR processes. The application features employee onboarding, attendance tracking, leave management, and payroll calculation, all wrapped in a secure and user-friendly interface.

View Code

Key Technologies

  • Frontend: Next.js (React) for a responsive and dynamic UI
  • Backend: Node.js & Express for robust API handling
  • Database: MySQL for structured employee data management
  • Auth: JWT-based authentication for secure role-based access

README.md

# Employee Management System ## Key Features - **Role-Based Access**: Separate portals for Admin and Employees. - **Attendance Tracking**: Geolocated check-in/out system. - **Payroll Automation**: Automatic salary calculation based on attendance. - **Leave Management**: Approval workflow for leave requests. ## Installation 1. Clone the repository ```bash git clone https://github.com/presha13/employee-management.git ``` 2. Install dependencies ```bash npm install ``` 3. Run the development server ```bash npm run dev ```

Project Impact

This solution digitized the client's manual HR operations, reducing payroll processing time by 40% and ensuring 100% accuracy in attendance records. It provided management with real-time insights into workforce availability and performance.
